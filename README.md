# Weighted Number Generator (1–80)

A web-based weighted random number generator where users select preferred numbers and influence probability outcomes using a custom weighting system.

---

## 🎯 Features
- Select numbers from 1–80
- Choose 20 preferred numbers
- Weighted probability generator
- Instant result display
- Mobile-friendly interface

---

## ⚙️ How It Works
The system assigns higher probability weight to selected numbers while maintaining randomness across the full 1–80 pool.

This creates a controlled random simulation experience.

---

## 💻 Technologies Used
- HTML
- CSS
- JavaScript

---

## 🚀 How to Use
1. Open `index.html`
2. Select your preferred numbers (up to 20)
3. Click generate
4. View weighted result

---

## 📌 Purpose
This project demonstrates weighted probability systems and interactive UI logic for simulation-based applications.
